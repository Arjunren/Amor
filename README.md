**Amor en Flor** is a sophisticated, mobile-first e-commerce front-end designed for high-end floral boutiques. It blends traditional serif elegance with modern web technologies to provide a seamless shopping experience.

---

## ✨ Key Features

### 🎨 Elegant Visual Design
* **Branded Palette:** Utilizes a custom-curated color scheme featuring `rose` (#D3968C) and `pale` (#F7FAD5) for a soft, professional aesthetic.
* **Typography:** Leverages high-contrast serif headers to establish a premium brand identity.
* **Fully Responsive:** Optimized for all devices, from mobile handsets to ultra-wide monitors.

### 🎠 Interactive Components
* **Hero Fade Swiper:** A smooth, auto-playing transition system for primary branding imagery.
* **Parallax Testimonials:** An immersive customer feedback section using Swiper.js parallax effects for depth and engagement.
* **Dynamic Navigation:** A responsive mobile menu system with smooth transition states.

### 🛍️ Smart Product Catalog
* **Real-time Search:** Instantly filter floral arrangements by title as you type.
* **Advanced Sorting:** Sort products dynamically by price (High-to-Low or Low-to-High) without page reloads.
* **JSON-ready Rendering:** Built to easily integrate with back-end APIs by rendering product grids from JavaScript objects.

## 🚀 Getting Started

### Prerequisites
No build step is required. The project uses Tailwind CSS via CDN for maximum portability.

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Arjunren/Amor.git
    cd Amor
    ```

2.  **Open in Browser:**
    Simply open `index.html` in your preferred web browser to view the site.

## ⚙️ Project Structure

The project is organized into modular HTML templates for easy maintenance:

| File | Purpose |
| :--- | :--- |
| `index.html` | Home page featuring hero transitions and customer feedback. |
| `products.html` | The main shop interface with search and sort logic. |
| `about.html` | Brand story and local sourcing information. |
| `contact.html` | Customer inquiries page with integrated Google Maps API. |

---

## 🧪 Technical Implementation

* **Styling:** Custom Tailwind CSS configurations for extended color palettes and serif font families.
* **Interactivity:** Vanilla JavaScript handles all DOM manipulations, including mobile menu toggles and product filtering logic.
* **Animations:** Swiper.js (v11) provides high-performance hardware-accelerated transitions for sliders.
